# 快速上传到 GitHub 指南

由于终端环境限制，请按照以下步骤手动上传：

## 方法一：使用 GitHub Desktop（最简单，推荐）

1. **下载 GitHub Desktop**
   - 访问：https://desktop.github.com/
   - 下载并安装

2. **打开项目**
   - 打开 GitHub Desktop
   - 点击 File → Add Local Repository
   - 选择项目目录：`C:\Users\Administrator\Desktop\遵纪守法好公民`

3. **发布到 GitHub**
   - 点击 Publish repository
   - Repository name: `Mobile-TimeTrigger-`
   - Owner: `jsc252323`
   - 勾选 "Keep this code private"（如果需要）
   - 点击 Publish repository

## 方法二：使用 Git Bash（如果已安装 Git）

1. **打开 Git Bash**
   - 在项目目录右键 → Git Bash Here

2. **执行以下命令**：

```bash
# 初始化仓库
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit: SmartMobileAutomation - 通用 Android App UI 自动化框架"

# 添加远程仓库
git remote add origin https://github.com/jsc252323/Mobile-TimeTrigger-.git

# 设置分支
git branch -M main

# 推送（会提示输入用户名和 Token）
git push -u origin main
```

**重要：** 推送时会要求输入：
- Username: 你的 GitHub 用户名
- Password: **使用 Personal Access Token**（不是账户密码）

## 方法三：使用 GitHub CLI（如果已安装）

```bash
# 初始化
gh repo create jsc252323/Mobile-TimeTrigger- --public --source=. --remote=origin --push
```

## 生成 Personal Access Token

如果使用方法二，需要生成 Token：

1. 访问：https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 设置：
   - Note: `Mobile-TimeTrigger-Upload`
   - Expiration: 选择合适的时间（如 90 days）
   - 勾选权限：**repo**（全部）
4. 点击 "Generate token"
5. **立即复制 Token**（只显示一次）

## 如果仓库已存在内容

如果远程仓库已有内容，需要先合并：

```bash
# 先拉取
git pull origin main --allow-unrelated-histories

# 解决冲突（如果有）
# 然后再次推送
git push -u origin main
```

## 推荐：使用 SSH（避免每次输入密码）

1. **生成 SSH 密钥**（如果还没有）：
```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
```

2. **复制公钥**：
```bash
cat ~/.ssh/id_ed25519.pub
```

3. **添加到 GitHub**：
   - 访问：https://github.com/settings/keys
   - 点击 "New SSH key"
   - Title: `My Computer`
   - Key: 粘贴刚才复制的公钥
   - 点击 "Add SSH key"

4. **更改远程 URL**：
```bash
git remote set-url origin git@github.com:jsc252323/Mobile-TimeTrigger-.git
```

5. **推送**：
```bash
git push -u origin main
```

## 需要帮助？

如果以上方法都失败，请检查：
1. Git 是否已安装：`git --version`
2. 网络连接是否正常
3. GitHub 账户是否有权限访问该仓库
4. 仓库名称是否正确：`Mobile-TimeTrigger-`
